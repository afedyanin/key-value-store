# Практикум: Пишем асинхронный TCP-сервер для кэша

## Цели занятия

- интегрировать знания о сокетах, пулах памяти и каналах в единое рабочее приложение;
- разрабатывать с нуля ядро высокопроизводительного сетевого сервера;
- применять на практике лучшие подходы к обработке клиентских подключений и асинхронной обработке данных.

## Краткое содержание

- Live-coding сессия по созданию TCP-сервера;
- прием подключений в цикле (AcceptAsync);
- чтение данных из сокета с использованием ArrayPool<T>.
- передача полученных данных в Channel для дальнейшей обработки;
- обработка отключения клиента и освобождение ресурсов.

## Результаты

- каркас асинхронного TCP-сервера, принимающего подключения и читающего данные.
