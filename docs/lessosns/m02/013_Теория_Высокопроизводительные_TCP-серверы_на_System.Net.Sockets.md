# Теория: Высокопроизводительные TCP-серверы на System.Net.Sockets

## Цели занятия

- понимать архитектуру асинхронных сетевых серверов;
- выбирать подходящую модель для работы с сокетами (async/await vs SocketAsyncEventArgs);
- анализировать типичные ошибки при разработке сетевых приложений (неправильная обработка буферов, утечки ресурсов).

## Краткое содержание

- класс System.Net.Sockets.Socket;
- модель Accept-Receive-Send;
- асинхронные методы сокетов (ReceiveAsync, SendAsync);
- продвинутая модель с SocketAsyncEventArgs для минимизации аллокаций;
- проектирование протокола прикладного уровня.
