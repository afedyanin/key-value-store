# Каналы (System.Threading.Channels): современный асинхронный producer/consumer

## Цели занятия

- реализовывать паттерн "Производитель-потребитель" с помощью Channels;
- строить эффективные асинхронные конвейеры для разделения I/O-операций и логики обработки;
- управлять жизненным циклом канала и обрабатывать его закрытие.

## Краткое содержание

- класс System.Threading.Channels.Channel<T>;
- ChannelWriter<T> и ChannelReader<T>;
- асинхронные операции WriteAsync и ReadAsync;
- типы каналов: ограниченные (bounded) и неограниченные (unbounded);
- сценарии использования: обработка запросов, фоновые задачи, логирование.

## Результаты

- реализация паттерна "Производитель-потребитель" с помощью System.Threading.Channels.Channel<T>.
