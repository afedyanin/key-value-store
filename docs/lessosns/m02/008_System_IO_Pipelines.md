# System.IO.Pipelines: высокопроизводительная передача и обработка потоков данных

## Цели занятия

- использовать System.IO.Pipelines для обработки потоков байтов в асинхронных приложениях;
- применять пайплайны для эффективного разделения ввода-вывода и логики обработки;
- управлять буфером и памятью при работе с большими объёмами данных;
- анализировать поведение пайплайнов в многопоточной среде и устранять узкие места.

## Краткое содержание

- Класс System.IO.Pipelines.Pipe
- интерфейсы PipeReader и PipeWriter;
- асинхронные методы: ReadAsync(), WriteAsync(), FlushAsync(), Advance();
- работа с ReadOnlySequence<byte>, Memory<byte> и Span<byte>;
- сценарии: сетевое взаимодействие, потоковая обработка данных, парсинг протоколов.

## Результаты

- реализация простого протокола обработки данных на основе PipeReader и PipeWriter, разделяющего ввод-вывод и логику обработки.
