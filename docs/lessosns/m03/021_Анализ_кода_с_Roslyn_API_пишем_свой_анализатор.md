# Анализ кода с Roslyn API: пишем свой анализатор

## Цели занятия

- использовать Roslyn API для программного анализа синтаксиса и семантики C# кода;
- разрабатывать собственные анализаторы кода для проверки корпоративных стандартов и поиска потенциальных ошибок;
- интегрировать анализаторы в процесс сборки для автоматизации code review.

## Краткое содержание

- .NET Compiler Platform (Roslyn);
- синтаксические деревья (SyntaxTree), узлы (SyntaxNode) и токены (SyntaxToken);
- семантическая модель (SemanticModel) для анализа типов и символов;
- создание проекта DiagnosticAnalyzer;
- практика: написание анализатора, который находит и подсвечивает проблемы в коде.

## Результаты

- проект Roslyn-анализатора с реализованной диагностикой определенного паттерна в коде.
