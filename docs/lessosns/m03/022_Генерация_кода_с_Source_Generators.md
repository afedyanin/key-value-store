# Генерация кода с Source Generators

## Цели занятия

- разрабатывать Source Generators для автоматического создания C# кода на этапе компиляции;
- устранять повторяющийся код (boilerplate) и заменять медленную рефлексию на сгенерированный код;
- применять метапрограммирование для повышения производительности и удобства разработки.

## Краткое содержание

- интерфейс ISourceGenerator;
- анализ кода пользователя с помощью Compilation и SyntaxReceiver;
- добавление сгенерированных файлов в процесс компиляции
- отладка Source Generators;
- практический пример: генерация INotifyPropertyChanged или паттерна Builder.

## Результаты

- простой Source Generator, генерирующий partial class на основе атрибута.
