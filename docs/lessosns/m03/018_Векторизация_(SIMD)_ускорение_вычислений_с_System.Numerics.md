# Векторизация (SIMD): ускорение вычислений с System.Numerics

## Цели занятия

- применять SIMD-инструкции для многократного ускорения математических операций над массивами данных;
- использовать тип Vector<T> для написания платформо-независимого векторизованного кода;
- определять, в каких задачах векторизация дает наибольший прирост производительности.

## Краткое содержание

- концепция SIMD (Single Instruction, Multiple Data);
- пространство имен System.Numerics.- Класс Vector<T> и операции над ним;
- проверка поддержки SIMD на целевой платформе;
- практический пример: ускорение обработки массивов чисел или пикселей.

## Результаты

- код, использующий Vector<T> для ускорения вычислений, и сравнительный анализ производительности.
