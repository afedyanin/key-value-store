# Практикум: Оптимизируем реальный код

## Цели занятия

- применять на практике весь цикл оптимизации: измерение, поиск узкого места, оптимизация, повторное измерение;
- комбинировать различные техники (работа с памятью, SIMD, алгоритмическая оптимизация) для достижения максимального результата;
- развить "чувство производительности" для интуитивного написания более эффективного кода.

## Краткое содержание

- Live-coding сессия по оптимизации.
- выбор неэффективного кода для анализа.
- Шаг 1: Написание бенчмарка (BenchmarkDotNet);
- Шаг 2: Профилирование и поиск узкого места;
- Шаг 3: Применение техник оптимизации;
- Шаг 4: Анализ результатов и выводов.

## Результаты

- оптимизированный фрагмент кода с результатами измерений "до" и "после".
