# Низкоуровневая работа с памятью: Span<T>, Memory<T>, stackalloc

## Цели занятия

- применять Span<T> и Memory<T> для обработки данных без лишних аллокаций и копирования;
- разрабатывать высокопроизводительные парсеры и форматеры для работы с бинарными данными и строками;
- использовать stackalloc для безопасного выделения небольших временных буферов на стеке.

## Краткое содержание

- Span<T> и ReadOnlySpan<T>: концепция "окна" в существующую память;
- Memory<T> и ReadOnlyMemory<T>: обертка над памятью, которую можно хранить в полях и передавать в асинхронные методы;
- операции среза (.Slice()) для работы с частями данных;
- безопасное использование stackalloc и его ограничения; практические паттерны: парсинг строк, работа с сетевыми буферами.

## Результаты

- код для парсинга данных с использованием Span<T>, демонстрирующий обработку без аллокаций.
