# Работа с неуправляемыми ресурсами: IDisposableAsync, SafeHandle, NativeMemory

## Цели занятия

- реализовывать паттерн IDisposable (+IDisposableAsync) корректно для детерминированного освобождения ресурсов;
- использовать SafeHandle для надежной и безопасной работы с нативными дескрипторами (файлы, мьютексы);
- применять NativeMemory для ручного выделения и освобождения неуправляемой памяти.

## Краткое содержание

- паттерн IDisposable (+IDisposableAsync) и using оператор;
- финализаторы (деструкторы) и их проблемы;
- класс SafeHandle как современная и надежная замена IntPtr и финализаторам;
- взаимодействие с нативным кодом (P/Invoke) и управление жизненным циклом нативных объектов;
- класс System.Runtime.InteropServices.NativeMemory для прямого управления памятью.

## Результаты

- реализация паттерна IDisposable с SafeHandle для безопасной работы с нативным ресурсом.
