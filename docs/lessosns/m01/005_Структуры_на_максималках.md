# Структуры на максималках: ref struct, readonly ref, in-параметры

## Цели занятия

- создавать высокопроизводительные типы данных с помощью ref struct для работы на стеке;
- предотвращать скрытое копирование больших структур с помощью in-параметров и readonly ref;
- разрабатывать API, сопоставимые по производительности с нативным кодом, оставаясь в безопасной среде C#.

## Краткое содержание

- модификатор ref для полей и локальных переменных.
- ref struct: типы, которые могут существовать только на стеке;
- readonly модификаторы для структур и их членов для гарантии иммутабельности;
- передача структур по ссылке: in, ref, out;
- практический пример: создание собственного типа Span-like.

## Результаты

- реализация ref struct с использованием in-параметров для предотвращения копирования и результаты измерения производительности.
