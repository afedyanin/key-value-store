# OpenTelemetry: современный стандарт наблюдаемости

## Цели занятия

- инструментировать приложения для сбора телеметрии (трассировки, метрики, логи);
- использовать OpenTelemetry для получения полного представления о работе распределенной системы;
- настраивать экспорт телеметрии в популярные системы мониторинга (Jaeger, Prometheus).

## Краткое содержание

- концепция наблюдаемости (Observability);
- три столпа: трассировки (Traces), метрики (Metrics), логи (Logs).- OpenTelemetry API и SDK;
- ActivitySource для создания трассировок;
- Meter для сбора метрик;
- инструментация популярных библиотек (ASP.NET Core, HttpClient).

## Результаты

- приложение, инструментированное с помощью OpenTelemetry, с добавленной трассировкой (Activity) и метрикой (Counter).
