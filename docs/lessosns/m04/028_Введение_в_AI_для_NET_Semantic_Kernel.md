# Введение в AI для .NET: Semantic Kernel

## Цели занятия

- понимать архитектуру и возможности Semantic Kernel для интеграции с большими языковыми моделями (LLM);
- создавать и вызывать "навыки" (skills) для решения задач с помощью AI;
- интегрировать LLM (такие как GPT) в свои .NET-приложения.

## Краткое содержание

- Microsoft Semantic Kernel SDK;
- концепции: ядро (Kernel), навыки (Skills), функции, коннекторы;
- создание промптов и вызов LLM;
- планировщики (Planners) для решения сложных задач;
- практический пример: создание чат-бота или суммаризатора текста.

## Результаты

- консольное приложение, использующее Semantic Kernel для вызова большой языковой модели.
