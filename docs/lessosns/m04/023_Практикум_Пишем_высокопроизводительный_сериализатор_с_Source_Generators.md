# Практикум: Пишем высокопроизводительный сериализатор с Source Generators

## Цели занятия

- применять метапрограммирование на этапе компиляции для создания высокопроизводительных компонентов;
- разрабатывать сложные Source Generators для автоматизации генерации контекстно-зависимого кода;
- заменять медленную рефлексию на сгенерированный код, чтобы устранить runtime-overhead и достичь максимальной производительности;
- закрепить на практике полный цикл разработки генератора: от анализа кода до его генерации и использования.

## Краткое содержание

- Live-coding сессия по созданию кастомного сериализатора.
- Задача: написать генератор, который реализует быстрый бинарный сериализатор для любого класса, помеченного атрибутом.
    - Шаг 1: Создание атрибута [GenerateSerializer];
    - Шаг 2: Анализ класса, помеченного атрибутом, с помощью Roslyn API (поиск публичных свойств);
    - Шаг 3: Генерация partial class с методом SerializeTo(Stream stream), который напрямую пишет байты свойств в поток;
    - Шаг 4: Сравнение производительности сгенерированного сериализатора со стандартным System.Text.Json с помощью BenchmarkDotNet.

## Результаты

- Source Generator, который создает высокопроизводительный бинарный сериализатор для любого класса, помеченного атрибутом.
