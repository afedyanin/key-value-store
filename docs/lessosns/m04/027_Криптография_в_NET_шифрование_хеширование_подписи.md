# Криптография в .NET: шифрование, хеширование, подписи

## Цели занятия

- применять криптографические API .NET для обеспечения конфиденциальности и целостности данных;
- выбирать подходящий алгоритм для конкретной задачи (шифрование, хеширование, цифровая подпись);
- реализовывать базовые сценарии защиты данных в своих приложениях.

## Краткое содержание

- System.Security.Cryptography API;
    - симметричное шифрование (AES);
    - асимметричное шифрование (RSA);
    - хэширование (SHA256) и HMAC для проверки целостности;
    - цифровые подписи (ECDsa).

## Результаты

- код, демонстрирующий шифрование/расшифрование данных с помощью AES и проверку целостности с помощью HMAC.
